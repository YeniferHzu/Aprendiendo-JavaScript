Clase 11

Ejercicio propuesto

Crea un sistema simple para registrar usuarios. Deberás seguir estos pasos:

Define una clase llamada Usuario que tenga las siguientes propiedades:
nombre: String
apellido: String
email: String
password: String

Agrega un método a la clase Usuario llamado mostrarInformacion que imprima en la consola la información del usuario en el siguiente formato:
Nombre: [nombre]
Apellido: [apellido]
[email]

Crea dos instancias de la clase Usuario con datos ficticios y utiliza el método mostrarInformacion para imprimir la información de ambos usuarios.

Agrega un método adicional a la clase Usuario llamado cambiarPassword que permita cambiar la contraseña del usuario.

Llama al método cambiarPassword en una de las instancias de usuario para cambiar su contraseña y luego muestra la información actualizada usando mostrarInformacion.
